# Evaluation Rubric

Score each criterion from 1 to 5.

| Criterion | Description |
|---|---|
| Accuracy | Information matches the original transcript |
| Completeness | Important points are not omitted |
| Owner extraction | Responsible people are identified correctly |
| Deadline extraction | Dates and deadlines are captured correctly |
| Decision clarity | Confirmed decisions are separated from suggestions |
| Hallucination control | No unsupported information is added |
| Usefulness | The output is clear and actionable |
| Human editing needed | Estimate whether low, medium, or high editing is required |

## Suggested Interpretation

- 5: Excellent
- 4: Good
- 3: Acceptable
- 2: Weak
- 1: Poor
